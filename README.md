# FAKE--URL--DETECTOR
Beginner Python project to detect fake url. 
This project is a beginner-friendly Fake URL Detector written in Python. It analyzes a URL based on:

Use of HTTPS

Presence of @ symbol

Presence of suspicious words like login, verify, account

Use of URL shorteners like bit.ly, tinyurl, t.co

Based on these checks, it assigns a risk score and classifies the URL as:

LEGITIMATE → likely safe

SUSPICIOUS → may be risky

LIKELY FAKE → likely phishing/fa
